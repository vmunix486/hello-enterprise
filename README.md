# Hello, World! Enterprise edition

This is a project that tries to make the classic "Hello, World!" program as difficult, slow, and bloated as possible.

Usually, there will be 3 versions, one I make, one that Claude makes, and one that Github Copilot makes.

If you are wanting to see the rules, go into RULES.md.

# Enterprise

Why is this <i>Enterprise</i>? Well, most companies nowadays use AI to just produce, the slowest, slop code on the planet for the things that they make. This slop code is usually really big and slow because, well if everybody using our program are using fast, modern computers, then we can remove support for all these old computers that nobody uses anymore. By now you probably get the gist of the project and why it's named that. I do have some strong opinions about the last statement since my everyday laptop is a Dell Latitude D610.

This project could be used as a good learning tool now that I think about it since writing dirty code can help identify how you can turn dirty code and make it faster, which can help teach people how to make other project faster.
